## задание
- есть реляционная БД MySQL со схемой (страна-город, язык по стране).
- Необходимо вынести все данные, которые запрашиваются часто, в Redis

## 1.Перечень необходимого установленного ПО

* **IntelliJ IDEA Ultimate**
* **Java 18**
* **DOCKER**
* **Workbench**  (либо любой другой сервис для администрирования mySql)

## использовано
- MAVEN
- liquibase
- hibernate
- mySql
- Redis
- Lombok


## реализовано
- через liquibase поднята БД MySQL в контейнере Docker
- ORM БД реализован через Hibernate
- реализован сервис получения данных из mySQL
- реализован сервис записи данных в Redis
- реализован сервис получения данных из Redis
- реализовано сравнение времени выполнения запросов в  mySQL и Redis




## особенности реализации
- при запуске проекта проверяется наличие необходимой схемы БД в mySql,  при ее отсутсвии liquibase выполняет скрипт ее создания.

## запуск на локальной машине
- запустить  Docker
- стартовать контейнер mySQL
- стартовать контейнер Redis 
- RUN

## отчет
- результат тестирования выводится в консоль.
- в виде (пример)
- Redis:	110 ms
- MySQL:	803 ms
